## Project Title
File Handling Utility

## Description
This project demonstrates basic file handling operations in Java.
The program allows users to create, read, write, and modify text files using Java I/O classes.

## Features
- Create a text file
- Write data into the file
- Read data from the file
- Modify (append) data to the file

## Technologies Used
- Java
- FileWriter
- FileReader
- BufferedReader

## File Structure
- FileHandlingUtility.java
- sample.txt
- README.md

## How to Run the Program
1. Open Command Prompt in the project folder
2. Compile the program:
   javac FileHandlingUtility.java
3. Run the program:
   java FileHandlingUtility

## Output
- The program creates a text file named sample.txt
- Displays file content in the console
- Appends new data to the existing file
